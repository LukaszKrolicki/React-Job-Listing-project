#### **JSX (JavaScript XML)**:

React components typically return JSX, which is a syntax extension of JavaScript. JSX allows you to write HTML-like code within your JavaScript.

**Example**:

```jsx
jsxSkopiuj kodconst App = () => {
    return (
        <div>
            <h1>Welcome to React!</h1>
        </div>
    );
};
```
